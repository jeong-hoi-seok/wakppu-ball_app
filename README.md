# 🫧 왁뿌볼 앱

왁뿌볼이 비싸서, 앱으로 직접 만들었습니다.

화면의 왁뿌볼을 손으로 누르면 **찌그러지고 부서지는** 느낌을 줍니다.  
스트레스 해소용 디지털 왁뿌볼이에요.

---

## 🚀 앱 실행

```bash
npm install
npm start
```

터미널에서 키를 눌러 실행합니다.

| 키 | 플랫폼 |
|:---:|:---:|
| `i` | iOS 시뮬레이터 |
| `a` | Android 에뮬레이터 |
| `w` | 웹 브라우저 |

또는 아래 명령어로 바로 실행할 수 있습니다.

```bash
npm run ios      # iOS
npm run android  # Android
npm run web      # Web
```
